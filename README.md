## 安装使用

```bash
yarn add mp-sdk
# 或
npm i --save mp-sdk
```

使用示例：

```js
const sdk = require('mp-sdk');

const cloud = sdk('appid', 'secret');

// appid、secret、access_token、grant_type 4个字段可以忽略不写
cloud.auth.code2Session({
  js_code: 'js_code'
}).then(result => {
  // code here
});
```
